# 🐑 消消乐 — 生日惊喜版

看似普通的水果消消乐，通关后触发隐藏生日彩蛋 🎂

## 🎮 玩法

- **三层堆叠**：卡片以不规则堆叠排列，选完上层才能看到下层
- **三连消除**：点击卡片放入底部槽位，集齐 3 张相同水果自动消除
- **三关渐进**：每关难度递增，通关收集一个数字
- **终极惊喜**：集齐 6·2·2 三关数字后，触发生日蛋糕和祝福！

## 🎯 关卡设计

| 关卡 | 卡片数 | 消除组数 | 收集数字 | 堆叠层数 |
|------|--------|----------|----------|----------|
| 第一关 | 12 张 | 4 组 | 6 | 2 层 |
| 第二关 | 24 张 | 8 组 | 2 | 3 层 |
| 第三关 | 36 张 | 12 组 | 2 | 3 层 |

水果主题（与生日无关，制造最终反差）：

## 🚀 运行

双击 `index.html` 即可在浏览器中打开，无需任何依赖。

## 🛠 技术栈

- HTML + CSS + Vanilla JavaScript
- 零依赖，纯静态
- 响应式：手机 / 桌面均可
- 原生 DOM 堆叠引擎 + 遮挡检测
